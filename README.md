# Projeto de Modelagem: Previsão de Preços de Casas

## Dataset

**House Prices - Advanced Regression Techniques**

- **Fonte**: https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data
- **Tamanho**: 1460 observações × 80 features
- **Tipo**: Dados imobiliários de Ames, Iowa (EUA)
- **Variável-alvo**: `SalePrice` (preço de venda em dólares)

### Como Executar

```bash
# Instale as dependências
pip install -r requirements.txt

# Abra o notebook
jupyter notebook main.ipynb
```
